## ⚙️ Installation

### For Linux

```bash
sudo apt install python3 git -y
git clone https://github.com/Ch4120N/Charon-SMS-Bomber.git
chmod 755 -R Charon-SMS-Bomber
cd Charon-SMS-Bomber
python -m pip install -r requirements.txt
python chsmsbomber.py
```
